Detailed Description

Library for uBlox SARA R410M LTE Cat M1/NB1 module.

Author
Bergma (Mike & Anders)
Date
2022-11

Use this library to initialize the module, setup the APN, connect to the network, and communicate via MQTT.

Steps to use this library:

  1. Call initModule() to initialize the module and enable AT interface and Timezone update
  1. Call setAPN() to set the operator APN
  1. Call getNetwork() to get status of network aquisition
  1. Call printInfo() to print connection information (TODO)
  1. Call setCertMQTT() to import certificates (If using SSL/TLS). This function is called 3 times, once for each certificate (CA, CERT, KEY)
  1. Call enableSSL() to enable SSL/TLS
  1. Call setMQTT() to set broker hostname and port
  1. Call willconfigMQTT() to set Last Will topic
  1. Call willmsgMQTT() to set Last Will message
  1. Call setMQTTtimeout() to set MQTT timeout
  1. Call setMQTTping() to set MQTT keepalive interval
  1. Call enableMQTTkeepalive() to enable MQTT keepalive
  1. Call loginMQTT() to login to MQTT broker
  1. Call publishMessage() to publish message to MQTT broker
