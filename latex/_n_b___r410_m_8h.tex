\doxysubsection{Detailed Description}
Library for u\+Blox SARA R410M LTE Cat M1/\+NB1 module. 

\begin{DoxyAuthor}{Author}
Bergma (Mike \& Anders) 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2022-\/11
\end{DoxyDate}
Use this library to initialize the module, setup the APN, connect to the network, and communicate via MQTT.

The function \mbox{\hyperlink{_n_b___r410_m_8h_a46a4e6cfc8eae8ca667e05b6b55f97c1}{print\+To\+Console()}} must be populated with the device specific implementation of Serial Print.

This library is based on the u-\/blox SARA-\/\+R4 AT Commands Manual (UBX-\/17003787 -\/ R09)
